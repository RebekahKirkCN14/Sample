To run this project you will need to install the following:
	Azure function tools - https://www.npmjs.com/package/azure-functions-core-tools
	SQL Server Express - https://www.microsoft.com/en-gb/sql-server/sql-server-downloads
	Postman - https://www.postman.com/downloads/

1. Open a local server through SQL Management studio.
2. Create a new SQL database on the local server named XXXX.
3. Use the migration scripts to create the table and stored procedure in SQL.
4. Use command window to clone the git repository and download a local copy of the solution with its dependecies. 
5. CD into the project folder in the command window and use FUNC start to run the solution.
